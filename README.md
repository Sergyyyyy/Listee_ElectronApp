# To-Do List

A lightweight task management application built with HTML, CSS, and JavaScript that allows users to create, edit, delete, and manage their daily tasks. The application stores data using the browser's Local Storage, ensuring tasks persist even after closing the application.

---

## 📖 Overview

This project is a simple yet functional To-Do List application developed to strengthen my understanding of JavaScript fundamentals, DOM manipulation, asynchronous programming, and client-side data persistence.

Rather than relying on external frameworks or databases, the application is built entirely with vanilla JavaScript. It demonstrates how dynamic user interfaces can be created by generating HTML elements through JavaScript while maintaining application state using Local Storage.

The project also served as practice for implementing CRUD (Create, Read, Update, Delete) operations and organizing JavaScript into reusable functions.

---

## ✨ Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete individual tasks
- 🧹 Clear all tasks with confirmation
- 💾 Automatically save tasks using Local Storage
- 🔄 Automatically restore tasks when the application starts
- 🌙 Toggle between Light Mode and Dark Mode
- ⚠️ Confirmation dialogs using SweetAlert2
- ⚡ Dynamic rendering without page refresh
